TEMA04

19/10/20 - 20/11/20

I tema 4 blev vi introduceret til programmeringssproget javascript, som bruges til at kode mere avancerede og dynamiske websites. Vi brugte javascript sammen med css animationer, til at kode vores egen spil. For at danne en struktur for vores spil brugte vi statemachine- og aktivitetsdiagram. Vi blev også her introduceret for illustrater hvor vi designede vores karakterer til spillet udfra skitser, moodboards og papirprototype. Derefter kodede jeg i html, css og javascript hvor jeg benyttede mig af funktioner, metoder som addClass/removeClass, variabler, events som load, click, animationend og betingelser (if-else).
